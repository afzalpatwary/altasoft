Bronze

Customer is provided with WordPress as their CMS solution. With this
package they can build a simple representation website

In the bronze packages customers obtains

  • 1 website, two sub domains
  • 100MB webspace for their website
  • 50 Mailboxes
  • 20 mailing lists
  • Unlimited bandwidth
  • 1-click WordPress Install
  • 10 WordPress Templates
  • Free SSL for the first year
  • Unlimited FTP Accounts
  • 10 Databases
  • WordPress Essentials Pack installed

Overuse policy: Overuse is not allowed.

  • The maximum disk usage is 100MB. At 75MB a message will be sent that the customer is reaching its limit. Maximum amount of traffic per month is unlimited.
  • The customer can have one domain and 2 subdomains. No Domain aliases are allowed.
  • A limit of 50 mailboxes is provided with a maximum size of 100MB each. In addition, the customer gets 20 mailing lists. Only WebMail is provided via the AfterLogic WebMail Light interface.
  • The interface for the WebMail will need to be adjusted to AltaSoft Hosting colours and house style.
  • The number of FTP accounts is unlimited.
  • The number of databases is 10.
  • One WordPress site can be installed per Bronze package and one weekly backup is provided. WordPress comes with Smart Update to keep WordPress on the right level of security

The WordPress Essential pack will come with the Bronze subscription. The
WP Essential Pack contains the following:

Security

  • Basic security options and brute-force protection.
  • Only Administrator users can see and manage this option.

Development Mode

  • Easy toggling WP_DEBUG or make your website accessible for logged-in users only
  • Only Administrator users can see and manage this option

Login Form

  • Change the WordPress logo shown above the login form to your own brand´s logo.
  • Only Administrator users can see and manage this option.

Google Analytics

  • Implement Google Analytics using analytics.js or gtag.js

We expect to run a maximum of 150 customer running on one server with their presentation site. This would lead to a total disk usage of 150 x 100MB = 15,000 MB
